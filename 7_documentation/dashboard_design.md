# Dashboard Design – Power BI

## Design Approach
The dashboard is designed for business stakeholders with a focus on clarity and decision-making.

## Pages Overview
1. Home - Dashboard introduction with navigation.
2. Executive Overview – Key KPIs
3. Churn Trends – Time-based analysis
4. Customer Segmentation – Risk-based insights
5. Churn Drivers – Behavioral indicators
6. Insights and Recomendation - Actionable insights and business recommendations.

## Key KPIs
- Total Customers
- Churn Rate
- Retention Rate
- High-Risk Customers
- Average Tenure
- Customer Lifetime Value (CLV)

## Design Principles
- Minimal clutter
- KPI-first layout
- Consistent color coding
- Filter-based interactivity
